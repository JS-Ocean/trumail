 How to use

            

                Using the email verification API endpoint is pretty straightforward. All you need to do is perform a GET request on the main lookup endpoint using the following schema.
                

                    trumail.io/{format}/{email}
                

                Currently there are two formats supported:
                json and xml.
            

        

    

    

        

             What it does

        

            Trumail is a fast and accurate email address verification API written entirely in Go. It was built with the intention of providing software engineers and businesses with a simple and easy to use solution to bounced emails.
        

        

            Whether you're a developer writing an application that requires the registration of new
            valid users or a business with a large mailing list, Trumail can handle the heavy lifting of verifying the validity of your email addresses so that you can make a confident decision on whether or not to dispatch messages to a given
            address.
        

        

            We currently provide a real-time lookup API which allows you to make an instant informed decision on user validity. In the near future we hope to have Bulk Lookups fully implemented.
        

    

    

        

             Privacy Policy

        

            To protect you and your users contact information, we DO NOT store any lookup data on our own servers. This is a basic API service and we therefore have no database to store data in. All that is currently collected are request logs in order to debug issues
            with the API, all of which are pruned every 1,500 messages (according to Herokus documentation).
